You can find the deployed project at [Pure Retail](https://app.pure-retail.com/).

## Getting Started

Clone the Repo to your local machine

```bash
git clone https://github.com/LABS-EU3/shopping_cart_frontend.git

```

```bash
cd <project-directory>
npm i
npm start
```
